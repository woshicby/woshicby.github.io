# CSS 目录说明文档

本目录包含网站所有的样式文件，采用模块化设计，分为公共样式和页面特定样式。

## 文件结构

```
CSS/
├── common.css         # 公共样式文件，包含全局样式和通用组件样式
├── index.css          # 首页专用样式
├── pace-calculator.css  # 配速计算器页面专用样式
├── sports.css         # 体育页面专用样式
├── study.css          # 学习页面专用样式
├── style-comparison.css  # 样式比较器页面专用样式
├── tools-common.css   # 工具集公共样式
├── tools.css          # 工具集页面专用样式
├── touken-forge.css   # 锻刀模拟器页面专用样式
└── video.css          # 视频播放器页面专用样式
```

## 文件说明

### common.css

**核心样式文件**，包含以下内容：
- **全局样式**：重置样式、基础字体、颜色变量定义
- **响应式布局**：媒体查询断点设置
- **公共组件**：
  - 导航栏样式与响应式导航（汉堡菜单）
  - 页脚样式
  - 按钮、表单等基础组件样式
- **主题系统**：支持深色/浅色主题的CSS变量定义
  - `:root`中的浅色模式变量
  - `[data-theme="dark"]`中的深色模式变量
- **设计系统变量**：颜色、间距、边框圆角等统一变量

### 页面特定样式文件

每个HTML页面都有对应的CSS文件，命名与HTML文件保持一致：

- **index.css**：首页布局和组件样式
- **pace-calculator.css**：配速计算器的专用布局和交互元素样式，包含返回小工具页面按钮样式的引用说明
- **sports.css**：体育页面的内容展示样式
- **study.css**：学习页面的布局和内容样式
- **style-comparison.css**：样式比较器页面的对比视图和交互样式
- **tools-common.css**：所有工具页面共享的公共样式组件，包含返回小工具页面按钮的统一样式
- **tools.css**：工具集页面的卡片式布局样式
- **touken-forge.css**：刀剑乱舞锻刀模拟器的游戏化界面样式
- **video.css**：视频播放器控件和布局样式

## 设计规范

### 颜色系统

网站使用一致的颜色系统，主要通过CSS变量定义在`common.css`中，并支持主题切换：
- **浅色模式变量**（默认）：定义在`:root`选择器中
- **深色模式变量**：定义在`[data-theme="dark"]`选择器中

主要颜色变量：
- `--primary-color`：主色调，用于导航栏、按钮等重要元素
- `--secondary-color`：辅助色，用于强调和交互状态
- `--text-color`：主要文本颜色
- `--background-color`：背景色
- `--border-color`：边框颜色
- `--card-color`：卡片和容器背景色

### 响应式设计

采用移动优先的响应式设计策略，主要断点：
- `max-width: 768px`：平板和移动设备
- `max-width: 480px`：小屏手机

### 命名规范

- 类名采用小写字母和连字符（kebab-case）
- 组件类名使用语义化描述
- 布局相关类名使用描述性前缀（如`.nav-`, `.main-`等）
- 主题相关类名使用`.theme-`前缀

## 开发指南

1. **添加新页面样式**：
   - 为新页面创建对应的CSS文件
   - 在HTML中先引入`common.css`，再引入页面特定CSS
   - 复用`common.css`中定义的变量和通用样式
   - 确保支持主题切换，使用CSS变量而非硬编码颜色

2. **修改公共样式**：
   - 仅在`common.css`中修改全局样式和通用组件
   - 修改颜色时，同时更新浅色和深色模式下的变量值
   - 确保修改不会影响现有页面的布局
   - 遵循现有的样式结构和命名规范

3. **性能优化**：
   - 避免过度使用CSS选择器
   - 减少冗余样式
   - 合理使用CSS变量实现主题一致性
   - 考虑使用CSS预处理器（如需要）

## 主题切换支持

网站支持深色/浅色主题切换功能，通过以下方式实现：

1. **CSS变量定义**：在`common.css`中定义两套颜色变量
2. **主题切换逻辑**：由`JS/theme-toggle.js`控制
3. **主题应用**：通过在`<html>`标签上添加/移除`data-theme="dark"`属性实现样式切换

开发新功能时，请确保：
- 使用CSS变量而非硬编码颜色值
- 测试在两种主题下的显示效果
- 特殊样式元素在两种主题下都有合适的表现

## 注意事项

- 所有页面样式都依赖于`common.css`，确保在HTML中正确引入
- 响应式导航功能由`JS/navigation.js`实现，相关样式定义在`common.css`中
- 主题切换功能由`JS/theme-toggle.js`实现，相关样式变量定义在`common.css`中
- 修改样式时请考虑不同屏幕尺寸和主题模式下的显示效果
- 工具页面间共享的样式组件（如返回按钮）定义在`tools-common.css`中，其他工具页面可直接引用
