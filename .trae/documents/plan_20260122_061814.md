# 资源系统现实化修改计划

## 目标
修改资源系统，使其更贴近现实世界：
1. 矿产、食物、能源、稀有资源靠人口在对应资源格子采集
2. 科技、经济、军事靠消耗资源产生
3. 科技和军事与战争相关
4. 经济和文化与贸易和外交相关

## 主要修改内容

### 1. 资源采集机制重构
- **采集主体**：人口是资源采集的主体，每个资源格子的采集量与人口数量成正比
- **采集效率**：不同资源类型有不同的采集效率，受科技和经济属性影响
- **采集上限**：每个资源格子的采集量有上限，取决于资源类型和地形

### 2. 属性发展机制重构
- **科技发展**：消耗矿产、能源、稀有资源，与战争有关
- **经济发展**：消耗矿产、能源，与贸易和外交有关
- **军事发展**：消耗矿产、能源、稀有资源，与战争有关
- **文化发展**：消耗文化资源，与贸易和外交有关

### 3. 资源消耗机制重构
- **明确资源用途**：每种资源有明确的消耗途径
- **消耗比例**：根据属性类型和发展阶段调整消耗比例
- **短缺影响**：资源短缺时，对应属性发展受阻

### 4. 资源类型定位
- **矿产**：基础工业资源，用于经济和军事发展
- **食物**：人口生存基础，影响人口增长
- **能源**：动力资源，用于科技和军事发展
- **稀有资源**：高端资源，用于高级科技和军事发展
- **文化资源**：文化发展基础，影响外交关系

### 5. 战争与贸易关联
- **战争影响**：战争消耗大量军事资源，获胜方获得资源奖励
- **贸易影响**：经济和文化属性影响贸易效率，贸易可获得额外资源
- **外交影响**：文化属性影响外交关系，良好关系促进贸易

## 具体修改步骤

1. **修改资源生产机制**
   - 资源生产改为主要由人口在资源格子采集
   - 不同资源类型有不同的采集效率
   - 采集量受科技和经济属性影响

2. **修改属性发展机制**
   - 科技发展需要消耗矿产、能源、稀有资源
   - 经济发展需要消耗矿产、能源
   - 军事发展需要消耗矿产、能源、稀有资源
   - 文化发展需要消耗文化资源

3. **增强战争与资源的关联**
   - 战争消耗大量军事资源
   - 获胜方获得战败方的部分资源
   - 军事属性影响战争结果

4. **增强贸易与资源的关联**
   - 经济和文化属性影响贸易效率
   - 贸易可获得额外资源
   - 外交关系影响贸易可能性

5. **优化资源流动机制**
   - 资源在格子间流动更符合现实逻辑
   - 贸易可实现文明间资源流动

6. **更新UI显示**
   - 显示资源采集效率
   - 显示资源消耗途径
   - 显示战争和贸易对资源的影响

## 预期效果
- 资源系统更贴近现实世界
- 资源采集与人口直接相关
- 属性发展与资源消耗直接相关
- 战争、贸易、外交与资源系统深度关联
- 游戏体验更具策略性和真实感
