终于换了新电脑！从Windows阵营跳到Mac，记录一下这段过渡期的真实感受～

# 📦 开箱

4月9日到货！新MacBook Pro到手的那一刻，包装质感确实没得说。苹果的包装设计一直都很讲究，拆封的过程本身就是一种仪式感。

开机之后第一件事就是迁移数据，苹果的迁移助理确实好用，从旧电脑无线传输数据，虽然速度不算快但胜在省心。

# 🖥 macOS初印象

## ✨ 让我惊喜的地方

### 触控板手势

Mac的触控板手势是第一个让我觉得"回不去了"的功能：
- **三指上滑**：Mission Control，一览所有窗口
- **三指左右滑动**：在全屏应用间切换，丝滑
- **双指缩放/旋转**：图片和网页操作特别自然
- **四指捏合**：Launchpad，快速找应用

Windows触控板虽然也在进步，但Mac的手势体验确实领先一大截。

### Spotlight搜索

`Command + Space`呼出Spotlight，不只是搜文件那么简单：
- 快速计算：直接输入算式出结果
- 单位换算：货币、长度、温度随输随换
- 词典查询：中英文都能查
- 打开应用：比在启动台里找快多了

基本可以当半个效率启动器用。

### 电池续航

M系列芯片的续航是真的猛。日常写代码、浏览网页、看文档，一天下来电量还很充裕。以前用Windows笔记本出门必带充电器，现在Mac经常不用带。

### 终端体验

作为开发者，macOS自带Unix终端是一大优势：
- 原生支持zsh，开箱即用
- Homebrew包管理器太好用了，`brew install`一行搞定
- 和Linux环境高度兼容，部署开发环境比Windows省心很多
- 不用再折腾WSL了

### 系统动画和流畅度

macOS的动画效果非常细腻，窗口缩放、切换都很顺滑。整个系统的视觉一致性很好，不像Windows那样有些地方还是老样式。

## 😤 不习惯的地方

### 键盘快捷键

从Windows过来最大的障碍就是快捷键体系完全不同：
- `Ctrl` → `Command`：复制粘贴变成了`Cmd+C`/`Cmd+V`
- `Alt+F4`关窗口 → `Cmd+W`关标签页/`Cmd+Q`退出应用
- `Ctrl+Shift+Esc`任务管理器 → `Cmd+Option+Esc`强制退出
- 截图从Win+Shift+S变成了`Cmd+Shift+3/4/5`

前两周经常按错，肌肉记忆需要时间调整😅

### 窗口管理

macOS的窗口管理逻辑和Windows很不一样：
- **没有最大化**：只有全屏和绿色按钮的"适合内容大小"
- **关闭窗口≠退出应用**：点红叉只是关窗口，应用还在运行
- **分屏不够灵活**：Mac有分屏吸附，但只有左右二分屏和四分屏，没法自动三等分，Windows的拖拽分屏更灵活
- **多桌面逻辑**：Mission Control需要适应，不像Windows任务栏那么直观

目前还在适应原生窗口管理，暂时没装第三方工具，靠全屏和分屏勉强够用。

### 文件管理

Finder和资源管理器的差异：
- **没有剪切**：只能复制然后`Cmd+Option+V`移动，没有直接的`Ctrl+X`
- **路径栏**：默认不显示完整路径，需要手动开启
- **回车键**：在Finder里回车是重命名，不是打开文件！打开要用`Cmd+O`
- **删除键**：Mac的Delete是退格，要真正删除文件得用`Cmd+Delete`

### 右键菜单

Mac的右键菜单功能比Windows少很多，很多操作需要进菜单栏找。不过后来发现`Option`+右键会多出一些选项，算是个小技巧。

# 🔧 必装软件推荐

适应Mac之后，整理了一些必装软件：

| 软件 | 用途 | 说明 |
|------|------|------|
| Homebrew | 包管理 | Mac必备，装啥都方便 |
| VS Code | 代码编辑 | 微软出品，跨平台无缝切换 |
| LM Studio | 本地模型 | 跑本地大模型，配合Openclaw使用，M芯片推理速度不错 |

# 💡 给Windows转Mac的人的建议

1. **给自己两周适应期**：刚开始不习惯很正常，坚持用下去就好了
2. **多用快捷键**：打印一份Mac快捷键速查表贴在旁边
3. **善用Spotlight**：能搜到的东西就不要手动找
4. **别怕终端**：macOS的终端比Windows友好很多，多学几个命令效率翻倍
5. **试试本地跑模型**：M芯片跑LM Studio体验不错，配合Openclaw很方便
6. **了解`Option`键的妙用**：很多操作按住Option会有额外选项

# 🎯 总结

从Windows转到Mac，整体体验是**先痛苦后真香**。刚开始确实各种不习惯，快捷键按错、找不到设置、窗口管理抓狂……但适应之后，Mac的触控板手势、终端体验、续航和系统流畅度确实让人很难回去。

另外M芯片跑本地模型的能力也让我惊喜，用LM Studio加载模型配合Openclaw使用，推理速度完全可用，这算是Mac的一个意外加分项。

如果问我值不值得换？**值得**，但要做好适应期的心理准备。毕竟用了十几年的Windows肌肉记忆，不是说改就能改的～
