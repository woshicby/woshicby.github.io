# 🎯 首次筋膜刀体验记录

前几天发现附近有一家筋膜刀专门店，预约今天跑后去做个松解～

# 💰今日花费：
- **首次一小时体验**：194元
- **十次一小时卡**：1976元（无限期，青岛三店通用）
- **去程打车**：13.99元
- **回程打车**：9.47元

### 总计
- **松解体验**：194元
- **办卡**：1976元
- **交通**：13.99元 + 9.47元 = 23.46元
- **总计**：194元 + 1976元 + 23.46元 = 2193.46元

## 📅 3月8日（当天）

### 松解前
- **时间**：下午3点前
- **活动**：进行了30分钟的有氧运动

### 松解过程
- **时间**：下午3点至4点
- **部位**：腿部筋膜（包括两条腿的大腿和小腿各大肌肉群）

小哥很细致，一直调整力度！大腿相对还行，小腿爆炸紧。中途还无意间复现了24年澄迈儋州的抽筋状况。感觉之前24年PB那几场抽筋的原因就在这里，办了十次卡，以后周末长距离完都来松松～

### 松解后反应
- **立即反应**：小腿后侧明显酸痛
- **后续活动**：步行2.28公里 + 0.74公里去吃肯德基
- **饮水量**：2180毫升
- **其他反应**：尿多

## 📅 3月9日（第二天）

### 身体反应
- **大腿**：轻微酸痛
- **小腿**：酸痛有所减轻，但左小腿依旧费劲
- **皮肤**：表面出现小块淤青

### 活动情况
- **运动**：没做运动

### 饮水量
- **总量**：4660毫升
- **其他反应**：依旧尿多

## 📅 3月10日（第三天）

### 恢复情况
- **右小腿**：基本恢复正常
- **左小腿**：酸痛有所减轻，但依旧有些影响行动

### 饮水量
- **总量**：3890毫升

## 📅 3月11日（第四天）

### 饮水量
- **总量**：2930毫升

## 📅 3月12日（第五天）

### 恢复情况
- **右小腿**：基本无感
- **左小腿**：发力仍有轻微酸痛
- **左右大腿**：几乎无感

### 训练情况
- **强度课**：8×1km@458-506/90s
- **实际情况**：没跑到目标速度，8组均速535左右

## 💭 总结与观察

1. **恢复进程**：5天时间内逐渐恢复，右小腿恢复较快（第3天基本正常，第5天基本无感），左小腿恢复较慢（第5天仍有轻微酸痛），左右大腿恢复最好（第5天几乎无感）
2. **疼痛反应**：从松解后立即的明显酸痛逐渐减轻，第5天已基本可以正常训练
3. **皮肤反应**：第2天出现淤青，属于正常现象
4. **饮水情况**：松解后饮水量显著增加（第1天2180ml，第2天4660ml，第3天3890ml，第4天2930ml），尿量也明显增多
5. **活动影响**：第5天已能进行强度课训练（8×1km），虽未达到目标配速，但说明恢复效果良好

## 📋 后续计划

- 继续观察恢复情况
- 适当进行轻度活动促进血液循环
- 保持充足水分摄入
- 如症状持续或加重，考虑咨询专业人士
- 十次卡已经办好了！以后周末长距离完必来～
