# 刀剑乱舞锻刀模拟器 - 图片资源说明

## 目录结构

此目录用于存放刀剑乱舞锻刀模拟器所需的图片资源，包含以下内容：
- `default_sword.jpg`: 默认显示的刀剑图片
- `刀账图标/`: 存放各种刀剑角色的图标图片

## 文件说明

### 默认图片要求

请确保此目录下存在`default_sword.jpg`文件，作为锻刀模拟器的默认显示图片。

**图片规格要求：**
- **宽度**：150-200像素
- **高度**：200-250像素
- **格式**：JPG或PNG
- **文件大小**：建议不超过50KB，以确保页面加载速度

### 刀账图标说明

`刀账图标/`子目录用于存放各个刀剑角色的图标图片，建议按照以下规范组织：
- 图片命名：使用刀剑的日文名称或ID作为文件名
- 尺寸建议：统一为64x64像素的方形图标
- 格式：PNG格式，支持透明背景

## 技术实现

这些图片资源通过`touken-forge.html`页面中的JavaScript代码（位于`JS/touken-forge.js`）进行加载和显示。当用户在锻刀模拟器中进行操作时，系统会根据不同的结果展示相应的刀剑图片。

## 注意事项

- 确保图片质量清晰，同时控制文件大小以优化性能
- 建议定期备份此目录下的所有图片资源
- 添加新的刀剑图片时，请确保同时更新相关的JavaScript代码以支持新图片的加载
