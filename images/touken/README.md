# 刀剑乱舞锻刀模拟器 - 图片资源说明

## 目录结构

此目录用于存放刀剑乱舞锻刀模拟器所需的图片资源，包含以下内容：
- `default_sword.png`: 默认显示的刀剑图片
- `刀账图标/`: 存放各种刀剑角色的图标图片

## 文件说明

### 默认图片要求

请确保此目录下存在`default_sword.png`文件，作为锻刀模拟器的默认显示图片。

**图片规格要求：**
- **宽度**：150-200像素
- **高度**：200-250像素
- **格式**：推荐使用PNG格式以支持透明背景和主题切换
- **文件大小**：建议不超过50KB，以确保页面加载速度
- **主题适配**：确保在深色和浅色背景下都清晰可见，建议使用透明背景

### 刀账图标说明

`刀账图标/`子目录用于存放各个刀剑角色的图标图片，建议按照以下规范组织：
- 图片命名：使用刀剑的日文名称或ID作为文件名
- 尺寸建议：统一为64x64像素的方形图标
- 格式：推荐使用SVG或PNG格式
  - SVG格式可以通过CSS动态改变颜色，更适合主题切换
  - PNG格式需为每个主题提供单独版本
- **主题支持**：
  - `刀账图标/light/`: 存放浅色主题下显示的图标（深色图标）
  - `刀账图标/dark/`: 存放深色主题下显示的图标（浅色图标）
  - 如使用SVG格式，可只存放一份，通过CSS变量动态调整颜色

## 技术实现

这些图片资源通过`touken-forge.html`页面中的JavaScript代码（位于`JS/touken-forge.js`）进行加载和显示。当用户在锻刀模拟器中进行操作时，系统会根据不同的结果展示相应的刀剑图片。

### 主题切换支持

系统支持深色/浅色主题切换功能：
- 通过`JS/theme-toggle.js`控制整体主题切换
- `JS/touken-forge.js`会监听主题变化事件，动态调整图片资源路径
- 对于SVG图标，通过修改CSS颜色变量实现动态颜色切换，无需加载多个版本

## 注意事项

- 确保图片质量清晰，同时控制文件大小以优化性能
- 建议定期备份此目录下的所有图片资源
- 添加新的刀剑图片时，请确保同时更新相关的JavaScript代码以支持新图片的加载
- 为支持主题切换，新增图片资源时请同时提供深色和浅色主题版本
- 优先考虑使用SVG格式的图标，可减少资源数量并提高切换性能
- 确保默认图片在不同主题下都有良好的视觉效果和对比度
- 图片加载错误时，系统会回退到默认图片，请确保默认图片始终存在且可用
- 考虑为新添加的图片实现懒加载功能，提升页面初始加载速度

## 性能优化建议

- 对图片进行压缩优化，减小文件体积
- 使用CSS sprites技术合并小图标，减少HTTP请求
- 考虑使用WebP格式替代部分PNG图片，提高加载速度
- 实现图片懒加载，特别是角色和武器图片
- 对于频繁访问的图片，考虑使用localStorage进行缓存
