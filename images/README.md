本目录用于存放网站所需的图片资源文件，包括页面元素、背景、图标和其他视觉内容。

> **注意**：`images/touken`子目录有其独立的说明文档`README.txt`，请参考该文档了解刀剑乱舞相关图片资源的规范。


## 推荐文件格式

为了确保兼容性和良好的加载性能，建议使用以下图片格式：

- **JPEG (.jpg)**：适用于照片和色彩丰富的图像
  - 推荐压缩质量：80-90%
  - 适用场景：背景图片、照片展示

- **PNG (.png)**：支持透明背景，无损压缩
  - 适用场景：图标、Logo、需要透明背景的图片

- **WebP (.webp)**：现代图片格式，压缩率更高
  - 提供与JPEG/PNG相当的视觉质量，但文件更小
  - 考虑提供JPEG/PNG作为降级备选

- **SVG (.svg)**：矢量图形格式
  - 适用场景：图标、简单图形、Logo（可缩放且文件小）
  - 特别适合主题切换，可以通过CSS动态修改颜色，减少资源加载

### 主题相关图片格式
- 对于需要在深色/浅色主题间切换的图标，推荐使用SVG格式
- 如使用PNG格式，需为每个主题提供单独版本并放在对应的目录中

## 命名规范

为了便于管理和使用，建议遵循以下命名规范：

- **小写字母**：文件名全部使用小写字母
- **连字符分隔**：单词之间使用连字符（-）分隔，例如：`profile-picture.jpg`
- **描述性命名**：文件名应清晰描述图片内容，例如：`homepage-banner.jpg`
- **前缀标识**：可以使用前缀标识图片类型，例如：
  - `bg-`：背景图片（background）
  - `icon-`：图标
  - `logo-`：标志
  - `btn-`：按钮图片

### 主题相关图片命名
- 对于需要主题适配的图片，可在文件名中添加主题标识
- 例如：`icon-menu-light.svg`, `icon-menu-dark.svg`
- 对于放在特定主题目录中的图片，可保持文件名一致，通过目录结构区分

## 图片优化指南

### 尺寸建议

- **桌面网站**：主要内容图片宽度建议为1200-1920px
- **响应式图片**：为不同屏幕尺寸提供多个版本
- **缩略图**：产品/项目缩略图建议为300-600px宽
- **图标**：尺寸建议为16x16px、32x32px、64x64px

### 性能优化

- 压缩图片文件大小，在视觉质量和加载速度之间取得平衡
- 使用响应式图片技术（HTML5 `srcset` 和 `sizes` 属性）
- 实现延迟加载（lazy loading），只在需要时加载图片
- 考虑使用图片CDN服务加速加载

### 响应式图片策略

- 为同一图片提供多个尺寸版本
- 使用媒体查询或HTML5响应式图片语法
- 在移动设备上提供更小尺寸的图片以节省流量

## 使用建议

### 图片引用

在HTML中引用图片时，请使用相对路径：

```html
<img src="../images/your-image.jpg" alt="描述性替代文本">
```

### 背景图片

在CSS中使用背景图片时：

```css
.element {
  background-image: url('../images/background.jpg');
  /* 其他背景属性 */
}
```

### 无障碍考虑

- 为所有图片提供有意义的`alt`属性
- 对于纯装饰性图片，可以使用空`alt=""`
- 确保图片内容有足够的对比度

## 注意事项

- 确保所有图片的版权合法性
- 定期清理未使用的图片文件
- 备份重要图片资源
- 测试图片在不同设备和浏览器上的显示效果
